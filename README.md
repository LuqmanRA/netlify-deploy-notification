## Getting Started

First, run the development server:

```bash
npx auth secret
# and
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Second, run migration database:

```bash
 npm run db:migrate
```

Third, run seeder database:

```bash
 npm run db:seed
```

Runnig db studio:

```bash
 npm run db:studio
```
